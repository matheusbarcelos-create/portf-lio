Portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro, apresentando minhas habilidades, trajetória acadêmica e projetos como estudante de Engenharia da Computação.

---

## 🔗 Acesse o Projeto

> [matheusbarcelos.dev](https://matheusbarcelos-create.github.io/portf-lio/) ← substitua pelo link do seu deploy

---

## 📋 Seções

- **Início** — apresentação e chamada para contato
- **Especialidades** — linguagens, certificados e objetivos acadêmicos
- **Sobre** — trajetória pessoal e links sociais
- **Projetos / Portfólio** — trabalhos e experiências destacadas

---

## 🚀 Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura semântica das páginas |
| CSS3 | Estilização, responsividade e animações |
| JavaScript | Menu mobile e comportamento interativo |
| [ScrollReveal](https://scrollrevealjs.org/) | Animações de entrada dos elementos |
| [Bootstrap Icons](https://icons.getbootstrap.com/) | Ícones da interface |

---

## 📁 Estrutura de Arquivos

```
📦 portfolio
 ┣ 📄 index.html       # Página principal
 ┣ 📄 style.css        # Estilos globais e responsividade
 ┣ 📄 mobilemenu.js    # Lógica do menu mobile
 ┣ 📄 rev.js           # Configuração das animações ScrollReveal
 ┣ 📁 img/             # Imagens do projeto
 ┣ 📁 certificados/    # Arquivo .zip com certificados
 └ 📁 favicon_io/      # Ícones da aba do navegador
```

---

## 📱 Responsividade

O layout se adapta automaticamente para telas menores que **1224px**, com menu hamburguer, imagens redimensionadas e colunas empilhadas verticalmente.

---

## ✨ Funcionalidades

- Animações de entrada ao rolar a página (ScrollReveal)
- Menu mobile com abertura/fechamento suave
- Imagens com animação flutuante contínua
- Cards de portfólio com efeito hover
- Download direto dos certificados acadêmicos
- Links para redes sociais e WhatsApp
